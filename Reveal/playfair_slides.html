Digraph - message

Alice would like to send this message to Bob:

“secret messages”

Digraph - Rules

Alice's message must be processed through these following rules:

  1. Letters in the message must be split into PAIRS
  2. Separate duplicated letters by inserting X
  3. Insert X when there is an odd letter at the end of the message

Digraph - Processing

SECRET MESSAGES

SE CR ET ME SS A GE S

SE CR ET ME SX SA GE SX

Encryption - Rules

The encryption involves taking the pairs from digraph and process the following rules with the Table:

  1. IF in same row
    • Move to the right by one column
    • Wrap around the same row if end of row is reached
  2. IF in same column
    • Move down by one row
    • Wrap around the same column if end of column is reached
  3. IF it forms a rectangle
    • Swap the letters with the ones at the end of the rectangle
